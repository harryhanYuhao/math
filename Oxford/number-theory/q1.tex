This is the central problem of this mini-project
\begin{problem}
	Let $p$ be an odd prime and let $k \equiv 2 \mod 4$. 
	We can write $k = r^2d$ where $d$ is square-free. 
	We want to investigate the integer solution of the following equation
	\begin{equation}\label{eq:main}
	y^p = x^2 + k
\end{equation}
\end{problem}

In this mini-project, $\O$ will always denote the ring of integers of $\Qd$. 
Fraktur letters such as $\a, \b, \c$ will denote ideals in $\O$, while Latin and Greek letters such as $a, b, c$ and $\alpha, \beta, \gamma$ will denote elements in $\O$.
$\langle a \rangle$ denotes the principal ideal generated by $a$ in $\O$, and $\langle a, b, \cdots \rangle$ denotes the ideal generated by $a, b, \cdots$ in $\O$.
$h(\Q(\smd))$ denotes the class number of $\Qd$

\section{Q1}

In this section, we assume the class number of $\Q[\smd]$ is not divisible by $p$, and we only work on coprime solutions $x, y$.

\subsection{General algorithm for coprime solutions when \texorpdfstring{$h(\Q[\smd])$ is not divisible by $p$}{}}

The diopantine equation can be factorised in $\Qd$ as 

$$
	y^p = (x+ r \smd) (x - r \smd)
$$

Suppose a prime ideal $\p$ divides both of $\langle x + r \smd \rangle$ and $\langle x - r \smd \rangle$, then 
\begin{enumerate}
	\item $\p$ contains $x + r \smd + x - r \smd = 2x$;
	\item $\p$ contains $y^p$ therefore $\p$ also contains $y$.
\end{enumerate}
First notice that if $y$ is even, $x^2 = y^p - k$ must be even (as $k$ is also even), which implies $x$ is even, contradicting the fact $x$ is coprime to $y$. 
Same reasoning shows that $x$ can not be even either. 
Since discriminant of $\Qd = 4d$, 2 is ramified in $\Qd$ \footnote{2 is ramified means $\langle 2 \rangle = \q^2$ for some prime ideal $\q$ in $\O$. 
indeed, $\langle 2 \rangle = \langle 2, \sqrt{-d}\rangle$.}.
As a result, if $\p$ contains $\langle 2\rangle$ or the factor of $\langle 2 \rangle = \langle 2, \smd \rangle$, $\langle y^p \rangle$ must contain two copies of $\q$, making $y$ even, which is a contradiction.

As $\p$ does not contain $2$, it therefore must contain some prime ideal factors of $\langle x\rangle$, and therefore $\p$ contains $x$ as well.
Since $x$ and $y$ are coprime and $\p$ contains $y$ , $\p$ must be $\O$.
Therefore we conclude $gcd(\langle x + \smd \rangle, \langle x - \smd \rangle) = \O$ (they are coprime ideals).

Therefore by lemma \ref{lem:gcd-ideals}, we can write $\langle x + r\smd \rangle$ as $\mathfrak{a}^p$ for some ideal $\mathfrak{a}$ in $\O_{\Qd}$. 

Since $h(\Q[\smd])$ is not divisible by $p$, and $p$ is a prime number, $gcd(p, h(\Qd))=1$. 
Moreover, as $\a^p = \langle x + r\smd \rangle$ is principal, we see $\a$ must be principal as well by lemma \ref{lem:principal}.
We can write $\a = \langle \alpha \rangle$ for $\alpha = a + b\smd$, $a, b \in \Z$, 

Therefore, $\a^p = \langle \alpha^p \rangle$, which means
$(a + b\smd)^p = u(x + r\smd)$ for some unit $u$ in $\O_{\Qd}$. 
We know the unit group of $\O_{\Qd}$ is $\{\pm 1\}$ by lemma \ref{lem:unit}, 
therefore we conclude
\begin{equation}\label{eq:q1key}
(a + b\smd)^p = \pm(x + r\smd) = \pm x \pm r\smd
\end{equation}

We may assume the sign before $r\smd$ is positive. 
If it is not we can apply the above argument to its conjuate $x - r\smd$ and get the positive sign before $r\smd$.

Let us expand $(a + b\smd)^p$ using binomial expansion, we have
$$
	(a + b\smd)^p = a^p + \binom{p}{1} a^{p-1} b \smd + \binom{p}{2} a^{p-2} b^2 (\smd)^2 + \cdots + b^p (\smd)^p
$$
Note the term with $\smd$ can be written as 
\begin{dmath*}
	b\smd \left( \binom{p}{1} a^{p-1} + \binom{p}{3} a^{p-3} b^2 (\smd)^2 + \cdots + \binom{p}{p} b^{p-1} (\smd)^{p-1} \right) = b \smd f(a, b) 
\end{dmath*}
where $f(a, b)$ is some homogeneous polynomial of degree $p-1$ in $a$ and $b$.

Comparing with \ref{eq:q1key} we see 
\begin{equation}\label{eq:q1-homogeneous}
	b f(a, b) = r
\end{equation}
Since $b \in \Z$, it must be a divisor of $r$.
We can try all possible values of $b$ and check if there exists some integer $a$ such that $b f(a, b) = r$. 
Notice that when substituting $b$ into $f(a, b)$, $f$ becomes a polynomial of $a$ with integer coefficients. 
It is a result from elementary algebra that in such case all the integer roots of $f(a)$ (if any exists) are restricted into a finite set (see \autoref{lem:finite-integer-solutions}).
So we can use brute-force will suffice to check if there is any integer solutions.

\subsection{Coprime Solutions for \texorpdfstring{$y^3 = x^2 + 2$}{}}

In this case $d = 2$ and the class number of $\Q[\sqrt{-2}]$ is 1, which is not divisible by 3.

By our procedure above, we can write the equation as
\begin{align*}
	y^3 = (x + \sqrt{-2})(x - \sqrt{-2})
\end{align*}
our argument shows that
$$
	x + \sqrt{-2} = (a + b\sqrt{-2})^3
$$
for some $a, b \in \Z$.
Expanding the right hand side, we have
\begin{align*}
	(a + b\sqrt{-2})^3 &= a^3 + 3a^2 b \sqrt{-2} - 6ab^2 - 2 \sqrt{-2} b^3 \\ 
					   &= (a^3 - 6ab^2) + b(3a^2 - 2b^2) \sqrt{-2}
\end{align*}
That is $b(3a^2 - 2b^2) = 1$ for some $a, b \in \Z$.
Since $b$ is a divisor of 1, it can be $\pm 1$.
\begin{enumerate}
	\item $b = 1$ gives $3a^2 - 2 = 1 \implies a = \pm 1$
	\item $b = -1$ gives $3a^2 - 2 = -1 \implies 3a^2 = 1$ which has no solution.
\end{enumerate}
So only options are $b = 1, a = \pm 1$.
Plugin to origin equation gives the only coprime integer solutions to be $y = (1 + \sqrt{-2})(1 - \sqrt{-2}) =  3$ and $x = \pm 5$.

\subsection{Coprime Solutions for \texorpdfstring{$y^3 = x^2 + 18$}{}}\label{sol:coprime-18}

In this case we have $p = 3, d = 2, r = 3$ and the equation can be factorised as
$$
	y^3 = (x + 3\sqrt{-2})(x - 3\sqrt{-2})
$$
By similar arguments as above we can write 
$$
	x + 3\sqrt{-2} = (a + b\sqrt{-2})^3
$$
which gives $b(3a^2 - 2b^2) = 3$, and $b$ has to be $\pm 1, \pm 3$ which are the only divisors of 3.
\begin{enumerate}
	\item $b = 1$ gives $3a^2 - 2 = 3 \implies 3a^2 = 5$ which has no integer solution.
	\item $b = -1$ gives $3a^2 - 2 = -3 \implies 3a^2 = -1$ which has no integer solution.
	\item $b = 3$ gives $3a^2 - 9 = 1 \implies 3a^2 = 10$ which has no integer solution.
	\item $b = -3$ gives $3a^2 - 9 = -1 \implies 3a^2 = 8$ which has no integer solution.
\end{enumerate}
So there is no coprime integer solution for this equation.

\subsection{Coprime Solutions for \texorpdfstring{$y^7 = x^2 + 338$}{}}

Note $338  = 13^2 \cdot 2$, and $\O_{\Q{\sqrt{-2}}}$ is a PID.
Our factorisation gives 
$$
y^7 = (x + 13\sqrt{-2})(x - 13\sqrt{-2})
$$
Therefore we can write 
\begin{dmath}
x + 13\sqrt{-2} = (a + b\sqrt{-2})^7
=a^{7} + 7 \sqrt{-2}  a^{6} b - 42 a^{5} b^{2} - 70 \sqrt{-2}  a^{4} b^{3} + 140 a^{3} b^{4} + 84 \sqrt{-2}  a^{2} b^{5} - 56 a b^{6} - 8 \sqrt{-2}  b^{7}
\end{dmath}
which gives 
$$
b(7 a^{6} - 70 a^{4} b^{2} + 84 a^{2} b^{4} - 8 b^{6}) = 13
$$

\textbf{Case $b =1$}.
Our diophantine equation becomes
$$a^2(7 a^{4} - 70 a^{2} + 84)  = 21$$
Since $21 = 3\cdot 7$ is square free $a$ must be $\pm 1$, which indeed gives a valid solution.
And $y =(1+\sqrt{-2})(1+\sqrt{-2}) = 3$ and $x$ is $\pm 43$.

\textbf{Case $b =-1$}.
Our diophantine equation becomes
$$a^2(7 a^{4} - 70 a^{2} + 84)  = -5$$
Since $5$ is square free any $a$ must be $\pm 1$, which are not valid solution in this case.

\textbf{Case b =13}.
Our diophantine equation becomes
$$f(a) = a^2(7a^4 - 11830 a^2 + 2399124) = 38614471 = 7 \cdot 337 \cdot 16369 $$
So any solution $a$ must be $\pm 1$. 
Yet $f(1) = 2387301 $ so there is no solution in this case.

\textbf{Case b =-13}.
Our diophantine equation becomes
$$f(a) = a^2(7a^4 - 11830 a^2 + 2399124) = 38614473$$
Now, $38614485 = 3^2 \cdot 43 \cdot 113 \cdot 883$, so any solution $a$ must be $\pm 1$ or $\pm 3$.
But 
\begin{align*}
	f(1) &= 2387301 \\
	f(3) &= 20638989
\end{align*}
so there is no solution in this case.

In conclusion, the only coprime integer solutions for this equation are $y = 3$ and $x = \pm 43$.


\subsection{Bound on \texorpdfstring{$a, b, y$}{}}

In our deduction in the previous section we have shown that 
\begin{equation}\label{eq:lower-y}
	(a + b\smd)^p = \pm(x + r\smd)
\end{equation}
for some $a, b \in \Z$. 
Note that $y = (a + b\smd) (a - b\smd) = a^2 + b^2 d$.
If $b = 0$, \eqref{eq:lower-y} can not be satisfied since the right hand side has non-zero imaginary part.
So we can assume $b \neq 0$ implying 
$$
y \geq b^2d \geq d.
$$

We have also shown that 
$$
b f(a, b) = r
$$
where $f(a, b)$ is a homogenous polynomial defined as
$$
f(a, b) = \binom{p}{1} a^{p-1} + \binom{p}{3} a^{p-3} b^2 (\smd)^2 + \cdots + \binom{p}{p} b^{p-1} (\smd)^{p-1}
$$

Therefore $b$ is a divisor of $r$, implying
$$
-r \leq b \leq r
$$

Note that $f(a, b)$ is a factor of $r$ so $|f(a, b)|$ is bounded by $r$ as well. 
Since $-r\leq b \leq r$, we may assume $b$ is a constant. It is clear that $\lim_{a \rightarrow \infty} f(a, b) = \infty$, as the leading term of $f(a, b)$ is $p a^{p-1}$. 
So $a$ must be bounded as well.
Let as derive an explicit bound for $a$.

Regard $f(a, b)$ as a polynomial in $a$. 
For $a \geq 1$, we can supply the following generous bound

\begin{align*}
	f(a, b) - r&= \binom{p}{1} a^{p-1} + \binom{p}{3} a^{p-3} b^2 (\smd)^2 + \cdots + \binom{p}{p} b^{p-1} (\smd)^{p-1} - r\\
			&\geq p a^{p-1} - \binom{p}{3} a^{p-3} b^2 d - \cdots - \binom{p}{p} b^{p-1} d^{\frac{p-1}{2}} \\
			&\geq p a^{p-1} - \binom{p}{3} a^{p-3} b^2 d - \cdots - \binom{p}{p} b^{p-1} d^{\frac{p-1}{2}} - k\\
			&\geq p a^{p-1} - \binom{p}{3} a^{p-3} k^2 d - \cdots - \binom{p}{p} k^{p-1} d^{\frac{p-1}{2}} - k\\
			&\geq p a^{p-1} - \frac{p-1}{2} \binom{p}{\frac{p-1}{2}}k^{p-1} d^{\frac{p-1}{2}}  a^{p-3} 
\end{align*}

It is clear, if we want $f(a, b) - r < 0$, we must have
$$
a \leq \sqrt{\frac{p-1}{2p} \binom{p}{\frac{p-1}{2}}k^{p-1} d^{\frac{p-1}{2}}}= A(p, k)
$$

Our deduction above only relies on the value of $k$ and the homogeneous polynomial $f(a, b)$, which is determined by $p$. 
Therefore we conclude $A(p, k)$ is the bound for $a$.
Since $y = a^2 + b^2d$, we conclude 

$$
y \leq A(p, k)^2 + r^2 d \leq A(p, k)^2 +k^3 
$$

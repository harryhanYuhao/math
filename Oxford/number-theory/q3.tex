\section{Q3}

In this section we investigate what happens if $p | h(\Qd)$.

\subsection{Coprime Solutions}

Assuming $x, y$ are coprime solutions for the equation 
$$
	y^p = x^2 + r^2 d
$$
but this time $p | h(\Qd)$.
Assuming $C_1, C_2, \cdots, C_m$ are the ideal classes of $\Qd$ whose $p$-th power is the principal class.
For each ideal class $C_i$, we can find an ideal $\b_i$ which is the inverse of $C_i$.
Without loss of any generosity let $\a \in C_i$. 
We have $1 = ([\b_i \a])^p = [\b_i]^p[\a]^p = [\b_i]^{p}$.
This means $\b_i^p$ is a principal ideal.

Let us denote this set as $\mathfrak{B}:= \{\b_1, \cdots, \b_m\}$ and define 
$\mathcal{G} = \{\gamma_1, \cdots, \gamma_m\}$ where $\langle \gamma_i\rangle = \b_i^p$.

By the exact same reasoning as in the proof of question 1, we write
$$
	y^p = (x + r\sqrt{-d})(x - r\sqrt{-d})
$$
where $\langle x+ r\smd \rangle$ and $\langle x - r\smd \rangle$ are coprime ideals, and 
\begin{equation}\label{eq:q3-factor}
	\langle x + r\smd \rangle = \a^p
\end{equation}
for some ideal $\a$.
Since $p | h(\Qd)$, we can not conclude directly that $\a$ is principal.
However, we know $\a^p$ is a principal ideal so $\a \in C_i$  for some $i$.

Let $\b_i \in \mathfrak{B}$ be the inverse of $\a$.
This means $\c\a = \langle \delta \rangle$ for some $\delta \in \O$.
We have also shown that $\b_i^p = \langle \gamma_i \rangle$.
Therefore, multiplying both sides of \eqref{eq:q3-factor} by $\b_i^p$ gives 
$$
	\langle \gamma_i \rangle \langle x + r\smd \rangle = (\b_i\a)^p = \langle \delta \rangle^p
$$
which means 
$$
	\gamma_i (x + r\smd) = \delta^p
$$
This is our desired result.

\subsection{Non-coprime Solutions}

We adopte similar measure as in Q2 to tackle the non-coprime solutions case. 

As in Q2, if $x, y$ are non-coprime solutions with $gcd(x, y) = c$, we can write 
$$
	\alpha_i \langle x + r \smd \rangle = \h ^p
$$
for some ideal $\h$ and some constant $\alpha$ that depends only on the common factor $c$ (thus depend only on the factors of $r$).
Now, $\h^p$ is a principal ideal, so $\h$ is in some ideal class $C_j$.
Let $\b_i$ be the inverse of $C_j$ and $\gamma_j$ be the element such that $\langle \gamma_j \rangle = \b_i^p$
Then we have
$$
\alpha_i \gamma_j \langle x + r\smd \rangle= (\b_i \h)^p = \langle \delta \rangle^p
$$
for some $\delta \in \O$.
This means 
\begin{equation}\label{eq:q3-non-coprime}
	\alpha_i \gamma_j (x + r\smd) = \delta^p
\end{equation}
which is our desired result.

Using exactly the same argument as in \autoref{sec:homogeneous},
\autoref{eq:q3-non-coprime} can be transformed into an equivalent homogeneous equation on two variables called Thue equation, which can be effciently solved. 

Since there are only finitely many choices for $\alpha_i$ and $\gamma_j$ (in our deduction, $\alpha_i$ and $\gamma_j$ only depends on $r$ and the structure of $\Qd$),
we can solve all the corresponding Thue equations corresponding to \autoref{eq:q3-non-coprime} and check if its solution is indeed our desired solution for \autoref{eq:main}.

\subsection{Fault Tolerance with Multiple Code Blocks}

Here we discuss some more constructions on crystal code.

\subsubsection{CNOT Between Two Code Blocks}

CNOT between two code blocks is transversal and thus can be implemented with fault tolerance. 
This has been shown in 12.3 of \cite{kissinger}.

\subsubsection{Universal Single-qubit Logical Gates}

As shown in the previous section, we can implement $T$ gate on any logical qubit. 
Let $H_i$ denote the $H$ gate on the $i$-th logical qubit and $\vec{H}$ denote the simultaneous $H$ gates on all logical qubits. 
Then $\vec{H} T_i \vec{H}$ is $HTH$ gate on the $i$-th logical qubit as shown below.
\ctikzfig{T-HTH}
In the section 4.5.3 of \cite{Nielsen_Chuang_2010}, it is shown that $T$ and $HTH$ are a universal set of gate on one qubit. (They can approximate any one qubit gate with arbitrary precision.) 

Therefore, any one qubit Clifford gate can be implemented to arbitrary precision with fault tolerance, as we can implement simultaneous $H$ gate and $T$ gate wth fault tolerance.
As shown in the previous section, we can implement CNOT between any two logical qubits with fault tolerance.

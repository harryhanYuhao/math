\subsection{Clifford and T with Measurement}

Clifford + T computation are performed in the following three steps.
\begin{enumerate}
	\item Prepare the states $\ket{0}^{\otimes n}$,
	\item Perform a sequence of Clifford + T gates,
	\item Measure in $Z$ basis.
\end{enumerate}

We can inject the magic $T$ state into any qubit of our circuit with the following Pauli gadget.

\begin{align}\label{eq:embed-S}
	\tikzfig{q-2-3-1}
\end{align}

As shown in the the remark in section \ref{remark:alternating-pauli-clifford}, the gadget can be implemented with alternating Pauli measurements and Clifford gates on an extra ancilla qubit prepared in $\ket{T}$, where subsequent Pauli the Clifford gates only depend on the previous measurement outcomes.

Thus for a Clifford + T circuit, we can let the Clifford gates remain the same and replace each $T$ gate with the above Pauli measurement construction. 
As product of two clifford gate is still a clifford gate, the construction guarantees that there is a Pauli measurement between each clifford gate. 
We still need to make sure that no two Pauli measurements are next to each other. 
The only way this may take place is that there are two consective $T$ gates on the same qubit. 
However, two $T$ gates make an $S$ gate which is part of Clifford, so there is no such problem, and we conclude that Pauli measurement alternates with Clifford.

The number of $\ket{T}$ required is exactly equal to the number of $T$ gates in the original circuit.
In this way, we have simulated a Clifford + T circuit with alternating Pauli measurements and Clifford gates.

\begin{lemma}[Multiple of $\frac{\pi}{4}$ state]\label{lm:mul-Z-state}
	Using $k$ $Z$ measurements and $k$ $\ket{T}$, we can simulate the state $\ket{\frac{k\pi}{4}} $ in the exact same way as in lemma \ref{lm:Pauli-S}.
	Importantly, it means we can 
	\begin{enumerate}
		\item simulate a $\tikzfig{Z-state}$ with 8 $Z$ measurements and 8 $\ket{T}$,
		\item simulate a $\tikzfig{Z-Pi-state}$ with 4 $Z$ measurements and 4 $\ket{T}$.
	\end{enumerate}
\end{lemma}
